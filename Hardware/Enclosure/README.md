# Enclosure Design

This directory contains all mechanical enclosure design files for the FMCW Radar unit.

## Sub-directories

| Directory | Description |
|-----------|-------------|
| [`CAD/`](CAD/) | 3D CAD model files (e.g., `.step`, `.stl`, `.f3d`, FreeCAD projects) |
| [`Renders/`](Renders/) | Rendered images of the enclosure design |

## Design Notes

- The enclosure is designed to house the FMCW Radar PCB
- Provisions for antenna apertures and cable routing are included
- Suitable for 3D printing or CNC fabrication
